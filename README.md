# Pics By Marl

Landing page for **Pics By Marl** — a personal blog about wildlife and garden photography in Tilburg, Netherlands.

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
# Python
python -m http.server 8000

# Node (npx)
npx serve
```

## Gallery images

Place 12 gallery photos in the `assets/` folder as:

- `image1.png` through `image12.png`

See `assets/README.md` for the suggested order (butterfly on flower, Great Tit, flowers, dragonflies, insects, birds, lizard, blue tit).

## Tech

- HTML, CSS, Vanilla JavaScript
- Mobile-first, responsive
- SEO-friendly (meta, Open Graph, semantic HTML)
